Create a folder in which to place the jellyseer data.

You can create a first file named `docker-compose.yml` and put the following content in it

export const responseJellyseer = await fetch("https://raw.githubusercontent.com/gamingdy/My-HomeLab/main/jellyseer/docker-compose.yml");
export const contentJellyseer = await responseJellyseer.text();

<Code code={contentJellyseer} lang="yml" title="docker-compose.yml" />;

Once the configuration is complete, you can run the following command

```bash
docker compose up -d
```
