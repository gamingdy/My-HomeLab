Create a folder in which to place the lynx data.

You can create a first file named `docker-compose.yml` and put the following content in it

export const response = await fetch("https://raw.githubusercontent.com/gamingdy/My-HomeLab/main/lynx/docker-compose.yml");
export const content = await response.text();

<Code code={content} lang="yml" title="docker-compose.yml" />

After that you need to configuration your environnement variable

export const responseEnv = await fetch("https://raw.githubusercontent.com/gamingdy/My-HomeLab/main/lynx/.env");
export const contentEnv = await responseEnv.text();

<Code code={contentEnv} lang="yml" title="docker-compose.yml" />

export const responseDbEnv = await fetch("https://raw.githubusercontent.com/gamingdy/My-HomeLab/main/lynx/db.env");
export const contentDbEnv = await responseDbEnv.text();

<Code code={contentDbEnv} lang="yml" title="docker-compose.yml" />

Once the configuration is complete, you can run the following command

```bash
docker compose up -d
```
